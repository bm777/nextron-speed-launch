## Usage

### Clone the repo

```bash
git clone --branch js-tw https://github.com/bm777/nextron-speed-launch.git
```

### Install Dependencies

```
$ cd nextron-speed-launch

# using yarn or npm
$ yarn (or `npm install`)

```

### Use it

```bash
# development mode
npm run dev

# production build (see the command in the package.json to run for your specific arch)
npm run build
```
